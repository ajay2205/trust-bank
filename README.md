## Run Locally

Clone the project

```bash
  git clone https://github.com/ajay2205/trust-bank.git
```

clone the project in this directory of xampp folder

```bash
  xampp/htdocs
```

Start the xampp server

Paste the link in the browser
```bash
 localhost/trust-bank 
```
